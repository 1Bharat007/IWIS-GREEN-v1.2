# Roadmap

Our roadmap is structured transparently to show where the project stands and where it's heading.

---

## v1.0 — Completed

The foundation release includes:

- **AI Waste Scanner** — Gemini Pro Vision integration for real-time material classification
- **Citizen Marketplace** — Create listings with GPS coordinates for local recycler pickup
- **Recycler Geospatial Feed** — Haversine distance-sorted listing feed
- **Dynamic Pricing Engine** — Localized scrap price calculator (INR per kg)
- **Gamification** — Green Points, tier progression (Seed > Sprout > Tree), leaderboards
- **EcoBot AI Assistant** — RAG-enhanced chatbot with embedded knowledge base
- **Earnings Dashboard** — Interactive charts for CO2 impact and transaction history
- **JWT Authentication** — Role-based access control (Citizen / Recycler)
- **Notification System** — Server-generated alerts for listing state changes
- **Hotspot Map** — Leaflet-powered geospatial visualization
- **Production Deployment** — Render configuration with CI pipeline

---

## v1.1 — In Progress (Pilot Phase)

- **Jammu Field Pilot** — Onboarding local Kabadiwalas and 500+ citizens
- **PostgreSQL Migration** — Replace SQLite with managed PostgreSQL for persistent storage
- **UX Tuning** — Monitoring real-world outdoor usage on budget Android phones
- **PWA Support** — Service workers for offline scan queuing

---

## v1.2 — Planned

- **Multilingual Support** — Full Hindi (hi-IN) translation layer
- **SMS Integration** — Replace development OTPs with Twilio / MSG91
- **Image CDN** — Move thumbnails from Base64 to AWS S3 / Cloudinary
- **Advanced Analytics** — Municipal waste heatmaps and seasonal trend reports

---

## v2.0 — Future

- **Multi-City Expansion** — Region-specific pricing and recycler networks
- **Live Tracking** — Socket.io integration for real-time recycler ETA (Uber-style)
- **PostGIS** — Advanced geospatial queries (road distance, routing polygons)
- **Admin Dashboard** — Municipality-facing analytics and user management
- **Mobile App** — React Native or Expo for native Android/iOS experience
- **Carbon Credit Integration** — Blockchain-verified carbon offset certificates
