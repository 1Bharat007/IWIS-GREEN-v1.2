# IWIS Brand Identity

The IWIS brand identity is designed to communicate trust, sustainability, and technological sophistication. It leans heavily on modern open-source conventions popularized by tools like Vercel, Stripe, and Supabase.

## Typography

We use modern, highly legible sans-serif typefaces that render cleanly on digital displays.

*   **Primary Typeface (Headings):** `Inter` or `Outfit`
*   **Secondary Typeface (Body):** `Inter`
*   **Monospace (Code):** `JetBrains Mono` or `Fira Code`

## Color Palette

The color palette avoids gradients in UI elements and sticks to a minimal, high-contrast dark theme with vibrant green accents.

### Core Backgrounds
*   **App Background:** `#09090b` (Deep Slate / Zinc 950)
*   **Card Background:** `#18181b` (Zinc 900)
*   **Border / Subdued:** `#27272a` (Zinc 800)

### Accents (Sustainability & Action)
*   **Primary Accent:** `#4ade80` (Emerald 400)
*   **Primary Hover:** `#22c55e` (Emerald 500)
*   **Primary Subdued (Alpha):** `rgba(74, 222, 128, 0.1)`

### Typography Colors
*   **Primary Text:** `#f8fafc` (Slate 50)
*   **Secondary Text:** `#94a3b8` (Slate 400)
*   **Muted Text:** `#64748b` (Slate 500)

## Logo Usage
*   The SVG logo (`assets/logo.svg`) is the preferred format for all web and print uses.
*   The logo represents a synthesis of a digital circuit (AI) and a leaf (Sustainability) encased in a secure infrastructure shield.
